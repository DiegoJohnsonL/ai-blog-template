Welcome to the **AI Blog Template**. This post was written to demonstrate how
the blog works.

## How it works

This blog is powered by an AI content agent that you can talk to through
Telegram. Ask it to write a post, and it will:

1. Generate the content in MDX format
2. Commit it to the repository via the GitHub API
3. Trigger a deployment so the post goes live

## Managing translations

The same agent handles translations. Tell it to update a key and it will
modify the translation files directly in the repo.

real post!
