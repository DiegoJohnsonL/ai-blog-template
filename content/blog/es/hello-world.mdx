Bienvenido al **AI Blog Template**. Este post fue escrito para demostrar
cómo funciona el blog.

## Cómo funciona

Este blog está impulsado por un agente de contenido IA con el que puedes
hablar a través de Telegram. Pídele que escriba un post y él:

1. Generará el contenido en formato MDX
2. Lo commitea al repositorio a través de la API de GitHub
3. Dispara un deployment para que el post se publique

## Gestión de traducciones

El mismo agente se encarga de las traducciones. Dile que actualice una
clave y modificará los archivos de traducción directamente en el repo.

## ¿Qué sigue?

¡Prueba a enviar un mensaje al bot de Telegram y pídele que escriba tu
primer post real!
